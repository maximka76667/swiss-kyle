## Overview

This binary predates the Tauri app. It was used to submit `CutVideo` jobs manually via the command line during backend-only development. It is no longer present in the codebase — job submission is now handled entirely by the Tauri app (→ [[wiki/components/tauri-app]]).

Kept here for historical reference only.
